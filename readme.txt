== Description ==
Bitwords is the world's first advertising platform that is 100% transparent and based on the Ethereum blockchain.

This plugin enables WordPress users to easily integrate Bitword's content recommendation widgets on their site.
Generate direct revenue by surfacing Bitword's high quality sponsored content recommendations.

Disclaimer: This plugin connects to bitwords.io server to automatically register and collect information about your site and posts. We use this data to correctly serve the most relevant ads on your site. You can request us delete this information at any time by writing to us at contact@bitwords.io


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In the Menu section, you'll find the Bitwords icon. Click on it and revise through the Terms & Conditions.
4. Enter your Ethereum withdrawal address in the Payout section. This address is where Bitwords will start crediting you for all the ads that have been displayed on your site.



== Frequently Asked Questions ==
= Do I need to signup with Bitwords? Or sign any contract to start earning? =
No, just install the plugin and that's it. Your website will automatically get registered with Bitwords.

= How do I start earning from my site with this plugin? =
Just install the plugin and enter your Ethereum deposit address in payouts screen. Once activated, the plugin will start serving ads from Bitwords

= Do you support payouts in Bitcoin/Fiat? =
No, not really. Because all our funds are locked inside an Ethereum smart-contact, payout can only happen in Ether.
